HTML5

Introdução ao HTML5

O que é HTML5?

O Hypertext Markup Language (HTML) é uma linguagem de markup(marcação) utilizada na criação de páginas da web para ser lido por browsers.

Como funciona o HTML5?

Tags do HTML

O HTML utiliza elementos chamados de tags para o browser identificar o tipo de conteudo que se encontra dentro desses tags.

Exemplo de um tag

<title>Titulo da Pagina</title>

Os tags do HTML são definidos por um tag de abertura <title> (definido pelos simbolos < >) e um tag de fecho </title>(definido por </ >)sendo title o nome da tag que pretende utilizar.

O texto Titulo da Pagina que se encontra entre os tags é o conteudo da tag.

Para este exemplo,utilizamos a tag title que é utilizado para definir o titulo do documento(cabeçalho da página) do HTML e o conteúdo Titulo da Pagina vai ser o titulo a ser atribuido ao documento HTML.

Atributos de um tag

Os atributos da tag são informação adicional para um elemento(tag).Os atributos costumam vir num formato nome = "valor"

Todas as tags podem conter atributos especificos para esse tag e os atributos são inseridos dentro da tag de abertura a seguir ao seu nome.

Exemplo de um atributo

<p title="Sou um atributo titulo">

Esta parte dos atributos será abordado outra vez mais aprofundado no futuro.

Head e Body

Quando criamos um documento HTML é importante separar-mos o conteudo com as tags <head> e <body>

A tag <head> é utilizado para fornecer ao HTML informações metadata sobre o documento(informações de dados sobre outros dados), como por exemplo a tag <title> é uma tag utilizada dentro do head para indicar o titulo do documento(cabeçalho da pagina) do HTML.

A tag <body> é utilizado para exibir o conteudo na pagina da web, este tag é mais importante do que a head porque sem esta tag não seria visualizado nada na página.

Exemplo de um HTML com head e body

<head>
 <title>Titulo da Pagina</title>
</head>

<body>
 <h1>Exemplo de cabeçalho</h1>
 <p>Exemplo de paragrafo</p>
</body>

Estrutura simples de um HTML

Exemplo de um documento HTML

<!DOCTYPE html>
<html>

 <head>
  <title>Titulo da Pagina</title>
 </head>

 <body>
  <h1>Exemplo de cabeçalho</h1>
  <p>Exemplo de paragrafo</p>
 </body>

</html>

Neste exemplo, podemos visualizar uma estrutura básica do HTML, que envolve o exemplo anterior,mas contendo juntamente as tags <!DOCTYPE html> e a tag <html>.

A tag <!DOCTYPE html> serve para indicar ao browser que este documento trata-se de um documento do tipo HTML.

Esta tag é escrita no topo do documento e é só necessário escrever uma única vez no documento(não é necessário usar um tag de fecho).

A tag <html> serve para representar a raiz do html.

Esta tag é utilizada como recipiente para todo o conteudo do html (excepto o <!DOCTYPE html>) para o conteudo presente entre as tags <html> estar identificado como parte do html.
