Cores CSS

Especificamos cores css pelos nomes predefinidos ou por valores, RGB, HEX, HSL, RGBA ou HSLA.

Background Color

Podemos definir a cor que vai estar por trás de algo

Exemplo

<h3 style="background-color:blue;">Background-color Azul</h3>
<p style="background-color:green;">Background-color Verde</p>

Syntax CSS

A Syntax CSS divide-se em dois, ou seja para podermos editar um elemento html temos de ter um selector e uma declaração que se divide em propriedade e valor. Por exemplo:

h1 { color:blue;} aqui h1 é um selector, {color:blue;} é uma declaração, color é a propriedade, blue é o valor.

Selectores CSS

Exitem 4 tipos de Selectores em CSS

1-Selector de elemento

Exemplo

p {
text-align: center;
}

2-Selector por id

Exemplo

#div {
position:absolute;
text-align: center;
}

3-Selector por class

Exemplo

.div {
text-align: center;
}

4-Selector por grupo

Exemplo

h1,h2 {
color:blue;
}

Comentar código CSS

Para comentar linhas de código em CSS usa-se /* linha de código */

Exemplo

/* p {
text-align: center;
} */
