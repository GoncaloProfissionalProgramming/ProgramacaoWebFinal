CSS

Cores CSS

Especificamos cores css pelos nomes predefinidos ou por valores, RGB, HEX, HSL, RGBA ou HSLA.

Background Color

Podemos definir a cor que vai estar por trás de algo

Exemplo

<h3 style="background-color:blue;">Background-color Azul</h3>
<p style="background-color:green;">Background-color Verde</p>

Background-color Azul

Background-color Verde

Text Color

Podemos definir a cor do texto

Exemplo

<h3 style="color:blue;">Texto a Azul</h3>
<p style="color:green;">Texto a Verde</p>

Texto a Azul

Texto a Verde

Border Color

Podemos definir a cor do limite do texto

Exemplo

<h3 style="border: 2px solid blue;">Limite a Azul</h3>
<p style="border: 2px solid green;">Limite a Verde</p>

Limite a Azul

Limite a Verde
