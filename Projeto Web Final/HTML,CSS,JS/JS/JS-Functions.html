JavaScript

Functions JS

O que são Funções?

Uma função JavaScript é um bloco de código usado para executar uma tarefa específica. Uma função JavaScript é executada quando é chamada no código.

Exemplo

function soma(num1, num2) {
   return num1 + num2; // Retorna a soma de num1 com num2
}

Syntax de uma função

Para definir uma função necessitamos da palavra function seguida por um nome e parenteses "()".
Os nomes das funções podem conter letras, dígitos, sublinhados e cifrões.
Os parênteses podem incluir nomes de parâmetros separados por vírgulas: (parâmetro1, parâmetro2, ...)
O código a ser executado, pela função, é colocado dentro de chavetas "{}".
Os parametros são listados entre os parenteses.

Exemplo

function nome(parametero1, parametero2, parametero3) {
   // Código a ser executado
}

Chamar uma função

Para chamar uma função chamamos pelo nome, seguido de parenteses e se tiver parametros estes são inseridos dentro dos parenteses.

Exemplo

soma(1,2);

Resultado

3

Return da função

Quando o JavaScript encontra um return, a função parar de ser executada.
Se a função foi chamada a partir de uma instrução, o JavaScript "retornará" para executar o código após ser chamado.
Funções frequentemente calculam um valor de retorno. O valor de retorno é "devolvido" de volta a quem chama a função.

Exemplo

var x = soma(1, 2); // função é chamada function soma(a, b) { return a + b; // Retorna a soma de a com b }

Resultado

3
