Arrays JS

O que são Arrays?

Arrays são variáveis que podem guardar mais do que um valor. Se queremos por exemplo guardar dentro de uma variável todas as marcas de carros existentes no mercado podemos fazê-lo guardando os nomes das marcas dentro de uma só variável em vez de guardarmos uma marca por variavel. Exemplo em baixo:

Exemplo

var cars = ["Saab", "Volvo", "BMW", "Renault"];

Como criar um Array?

Um array tem a seguinte syntax:
var array_name = [item1, item2, ...];

Exemplo

var cars = ["Saab", "Volvo", "BMW"];

Como acessar a um elemento de um Array?

Acessamos um elemento de um array pelo seu index. O primeiro elemento de um array tem como index 0, o segundo elemento tem como index 1 etc...
Código para acessar a um elemento de um array:

Exemplo

var cars = ["Saab", "Volvo", "BMW"];
var name = cars[0];

Resultado

Saab

Como alterar o valor de um elemento de um Array?

Alteramos o valor de um elemento de um Array com o seguinte código:
cars[0] = "Opel";

Exemplo

var cars = ["Saab", "Volvo", "BMW"];
cars[0] = "Opel"; var name = cars[0];

Resultado

Opel
