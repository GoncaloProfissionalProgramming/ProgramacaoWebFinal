Variáveis e tipos de Dados JS

O que são Variáveis?

Numa linguagem de programação as variáveis são usadas para armazenar valores de dados. JavaScript usa a palavra-chave var para declarar variáveis. O sinal de igual é usado para atribuir valores a variáveis. Existem muitos tipos de dados para atribuir a variáveis. Por regra geral uma variável só pode ter um tipo de dados.

Tipos de Dados

Em JS existem sete tipos de dados que são:

  • Boolean (Verdadeiro ou Falso)
  • Null (valor vazio)
  • Undefined (não existe valor)
  • Number
  • String
  • Symbol
  • Object

Como declarar uma variável?

Para declarar uma variável pode-se fazer de duas formas, se for uma variável sem um valor definido ou seja, a variável só vai ser afetada mais tarde então declara-se da seguinte forma: var x;, se a variável for para ter um valor especifico antes de ser utilizada, por exemplo ser igual a 7 então declara-se desta forma: var x=7;.

Exemplo

var x; // sem valor definido
var z = 7; //variável z é igual a 7

Exemplos de variáveis

O valor das variáveis podem ser sempre alteradas. Alguns exemplos de variáveis:

Exemplo

var x; // sem valor definido
var z = 7; //Tipo de dados: number
var y= "Hello World" //Tipo de dados:String
var w= True //Tipo de dados:Boolean
var a= null //Tipo de dados:null
var b= undefined //Tipo de dados:undefined

Font Size

A propriedade CSS Font-Size é usada para definir o tamanho do texto.

Existem 3 formas de definir o tamanho do texto:

  • Percentagem (%)
  • Em (em)
  • Pixeis (px)

Exemplo

<p style="font-size: 120%;">Texto definido com Percentagem</p>
<p style="font-size: 5em;">Texto definido com em</p>
<p style="font-size: 25px;">Texto definido com pixeis</p>

Font Weight

A propriedade CSS Font-Weight é usada para definir o a grossura do texto.

Existem 3 formas de definir a grossura do texto:

  • Normal
  • Bold
  • Lighter
  • Por números

Exemplo

<p style="font-weight: normal;">Texto normal</p>
<p style="font-weight: bold;">Texto a bold</p>
<p style="font-weight: lighter;">Texto em lighter</p>
<p style="font-weight: 600;">Texto por numeros</p>
