Estruturas de Repetição JS

O que são estruturas de Repetição?

Estruturas de repetição são usadas para realizar ações até chegar a uma certa condição . Existem duas estruturas de repetição, o For e o While.

For

O for executa um bloco de código durante as vezes que forem precisas.

Exemplo

var i;
for (i = 0; i < cars.length; i++) {
   text += cars[i] + "<br>";
}

Resultado

text += cars[0] + "<br>";
text += cars[1] + "<br>";
text += cars[2] + "<br>";
text += cars[3] + "<br>";
text += cars[4] + "<br>";
text += cars[5] + "<br>";

While

O While executa um bloco de código enquanto uma condição não for igualada.

Exemplo

while (i < 10) {
   text += "The number is " + i;
   i++;
}

Resultado

The number is 0;
The number is 1;
The number is 2;
The number is 3;
The number is 4;
The number is 5;
The number is 6;
The number is 7;
The number is 8;
The number is 9;
