Operadores JS

O que são operadores?

Em JavaScript existem 4 tipos de operadores que são:

  • Operadores aritméticos
  • Operadores de atribuição
  • Operadores de comparação
  • Operadores de lógica

Operadores Aritméticos

São 8 os operadores Aritméticos:

  • + (Adição)
  • - (Subtração)
  • * (Multiplicação)
  • ** (Exponenciação)
  • / (Divisão)
  • % (Para descobrir o resto de uma revisão)
  • ++ (Incrementar)
  • -- (Decrementar)

Exemplo

var x = 3;
var z = 1;
var y = x + z; //isto dá 4
var y = x - z; //isto dá 2
var y = x * z; //isto dá 3
var y = x ** z;//isto dá 3 elevado a 1
var y = x / z; //isto dá 3
var y = x % z; //isto dá 0
var y = x++; //isto dá 4
var y = x--; //isto dá 2

Operadores de atribuição

São 6 os operadores de atribuição:

  • = (Igual)
  • += (Mais Igual)
  • -= (Menos Igual)
  • *= (Multiplicação Igual)
  • /= (Divisão Igual)
  • % (Resto Igual)

Exemplo

var x = 1;
var z = 3;
var z = x; //z é igual a 1
var z += x; //z é igual a 4(z=z+x)
var z -= x; //z é igual a 2(z=z-x)
var z *= x; //z é igual a 3(z=z*x)
var z /= x; //z é igual a 3(z=z/x)
var z %= x; //z é igual a 0(z=z%x)

Operadores de comparação

São 6 os operadores de comparação:

  • == (Igual)
  • === (Valor igual a )
  • != (Diferente de)
  • !== (Valor diferente a)
  • > (Maior que)
  • < (Menor que)
  • >= (Maior ou igual)
  • <= (Menor ou igual)

Exemplo

var z == x; //z igual a x
var z === x; //valor de z igual a x
var z != x; //z diferente de x
var z !== x; //valor do z diferente de x
var z > x; //z maior que x
var z < x; //z menor que x
var z >= x; //z maior ou igual a x
var z <= x; //z menor ou igual a x

Operadores de lógica

São 3 os operadores de lógica:

  • && (e)
  • || (ou )
  • ! (Não)

Exemplo

var z && x>10; //z e x maior que 10
var z || x; //z ou x
var z ! x; //Retorna True se for falso e falso se for True
