O que são Strings?

As strings em JavaScript são usadas para guardar e manipular texto. Uma string tem zero ou mais caracteres.

Strings

Exemplo

var x = "João";

Pode ter aspas ou plicas:

Exemplo

var nome = "João";
var nome2 ='António';

Pode-se usar aspas dentro de uma String, desde que as aspas não sejam iguais ás primeiras aspas:

Exemplo

var answer1 = "It's alright";
var answer2 = "He is called 'Johnny'";
var answer3 = 'He is called "Johnny"';

String Length

Para ver o comprimento de uma string usamos o método length.

Exemplo

var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
var sln = txt.length;

Strings podem ser objectos

Exemplo

var firstName = "John";
var firstName1 = new String ("John");

firstName1 é um objecto
firstName é uma string
