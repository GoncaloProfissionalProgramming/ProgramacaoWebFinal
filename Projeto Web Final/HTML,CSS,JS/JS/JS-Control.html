Estruturas de Controlo JS

O que são estruturas de Controlo?

Estruturas de Controlo são usadas para realizar determinadas ações dependendo de uma certa condição. Existem duas estruturas de controlo, o if e o Switch.

If

O If tem sempre uma condição, ou seja se 1 + 1 for igual a 2 então o código fará algo, mas se 1+1 não for 2 então o código irá fazer outra coisa.

Exemplo

if(1+1=2){
   return True;
}else{
   return False;
}

Switch

O Switch também tem uma condição e dependendo do resultado da condição ele vai andar a "saltar" até encontrar a case correcta.

Exemplo

switch(numero){
 case 1:
   return True;
   break;
 case 2:
   return False;
   break;
}
