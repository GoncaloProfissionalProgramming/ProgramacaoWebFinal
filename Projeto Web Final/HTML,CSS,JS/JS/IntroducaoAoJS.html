JavaScript

Introdução ao JS

O que é JS?

JS ou JavaScript é uma linguagem web que serve principalmente para programar o comportamento da página.

Como utilizar JS?

2 Formas de utilizar JS

1-Ficheiro externo por exemplo "script.js"

script.js

function myFunction(p1, p2) {
   return p1 * p2;
}
document.getElementById("demo").innerHTML = myFunction(4, 3);

2-Dentro de um ficheiro HTML usando a tag <script></script>

index.html

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<head>

<script>

function myFunction(p1, p2) {
   return p1 * p2;
}
document.getElementById("demo").innerHTML = myFunction(4, 3); </script>

</head>

<body>

<p id="demo"</p>
</body>
</html>

Prompt e Alert

Prompt

O método prompt() mostra uma caixa que pede ao user algo, por exemplo para escrever um nome.

Prompt

var person = prompt("Please enter your name");

Alert

O método alert() mostra uma caixa com uma mensagem especifica e um botão OK.

Alert

alert("Hello! I am an alert box!!");
