HTML List

Os elementos list(lista) permitem organizar linhas de texto em itens, consuante o utilizador definir.

Existem 2 tipos de lists:

<ul>, em que lista todos os itens da lista com pontos no ínicio.

<ol>, em que lista todos os itens da lista com numeração( o primeiro item tem o número 1,o segundo o número 2,etc.).

Ambos os tipos de lists utilizam a tag <li> para definir cada item.

List <ul>

Este tipo de list exibe os seus itens com pontos.

Exemplo de uma list ul (com resultado)

<ul>
 <li>B</li>
 <li>A</li>
 <li>C</li>
</ul>
  • B
  • A
  • C

List <ol>

Este tipo de list exibe os seus itens com numeros de uma forma ordenada(1,2,3,4,etc.).

Exemplo de uma lista ol (com resultado)

<ol>
 <li>B</li>
 <li>A</li>
 <li>C</li>
</ol>
  1. B
  2. A
  3. C
