HTML Heading

Os headings do HTML(cabeçalhos) são elementos usados com mais frequência para a criação de títulos e sub-titulos num texto HTML.

Estes tags são definidos por <hx> em que x é um número entre 1 a 6 ( <h1>, <h2>, <h3>, <h4>, <h5> e <h6>).

Esta tag tem de ser criado dentro do <body> do HTML para poder ser visualizado.

No HTML, quanto maior for o número do heading, menor será o tamanho do texto(por exemplo o heading <h1> tem o tamanho do texto maior que o heading <h2>).

O conteúdo dentro das tags do heading é o texto que será definido para o cabeçalho.

Exemplo de Headings diferentes(com resultados)

<h1>Exemplo de cabeçalho 1</h1>
<h2>Exemplo de cabeçalho 2</h2>
<h3>Exemplo de cabeçalho 3</h3>
<h4>Exemplo de cabeçalho 4</h4>
<h5>Exemplo de cabeçalho 5</h5>
<h6>Exemplo de cabeçalho 6</h6>
