HTML Image

O elemento image(imagem), definido por <img>, serve para colocar imagens no HTML para serem visualizadas.

O tag da imagem necessita de um atributo src com o valor igual ao URL(destinatário) onde a imagem se encontra (com raiz do destinatário a pasta em que o documento html se encontra).

Também poderá utilizar os atributosheight(altura) e width(largura) se quiser definir um tamanho especifico para a imagem.(Se não usar estes atributos a imagem terá o seu tamanho original).

Exemplo de uma imagem (com resultado)

<img src="analysisBlack16.png" width="50" height="50"></img>
Nota:

No src também poderá indicar um URL online de uma imagem e,desde que o utilizador tenha acesso á internet, será visualizada essa imagem.
