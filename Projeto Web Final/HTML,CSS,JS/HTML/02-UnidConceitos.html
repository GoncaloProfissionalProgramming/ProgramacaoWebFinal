Cabeçalhos

Os cabeçalhos do HTML são definidos pela tag <hx> em que x é um número entre 1 a 6 ( <h1>, <h2>, <h3>, <h4>, <h5> e <h6>).

quanto maior for o número do cabeçalho, menor será o tamanho do cabeçalho(por exemplo o cabeçalho <h1> tem o tamanho do texto do cabeçalho maior que o cabeçalho <h2>).

O conteudo entre as tags do cabeçalho é o texto que será definido para o cabeçalho.

Exemplo de Cabeçalhos diferentes(com resultados)

<h1>Exemplo de cabeçalho 1</h1>
<h2>Exemplo de cabeçalho 2</h2>
<h3>Exemplo de cabeçalho 3</h3>
<h4>Exemplo de cabeçalho 4</h4>
<h5>Exemplo de cabeçalho 5</h5>
<h6>Exemplo de cabeçalho 6</h6>

Parágrafos

Os paragrafos do HTML são definidos pela tag <p> e o conteudo entre os tags é o texto a ser definido para o paragrafo.

Exemplo de paragrafo (com resultado)

<p>Exemplo de um paragrafo </p>

Quebra de linha

As quebras de linha <br> são tags que servem para quebrar uma linha num texto

O texto escrito após esta tag será exibido noutra linha

Exemplo de uma quebra de linha num texto (com resultado)

<p>Exemplo de <br> uma quebra de linha </p>

Imagens

Poderá adicionar uma imagem ao html utilizando a tag <img>.

O tag da imagem necessita de um atributo src com o valor igual ao destinatário da imagem (com raiz do destinatário a pasta em que o documento html se encontra).

Também poderá utilizar os atributosheight(altura) e width(largura) se quiser definir um tamanho especifico para a imagem.(Se não usar estes atributos a imagem terá o seu tamanho original).

Exemplo de uma imagem (com resultado)

<img src="img/analysisBlack16.png" width="50" height="50"></img>

Botão

Os botões são definidos pela tag <button> em que o texto escrito entre as suas tags é o texto que será exibido no botão.

Nota: Os botões serão explicados com mais pormenor noutra secção.

Exemplo de um botão (com resultado)

<button>Texto do botão</button>

Listas

As listas permitem listar os seus itens.

O HTML possui 2 tipos de listas: <ul> e <ol>

Ambos os tipos de listas utilizam a tag <li> para definir o item a ser adicionado para a lista.

Listas <ul>

Este tipo de lista exibe os seus itens com pontos.

Exemplo de uma lista ul (com resultado)

<ul>
 <li>B</li>
 <li>A</li>
 <li>C</li>
</ul>
  • B
  • A
  • C

Listas <ol>

Este tipo de lista exibe os seus itens com numeros de uma forma ordenada(1,2,3,4,etc.).

Exemplo de uma lista ol (com resultado)

<ol>
 <li>B</li>
 <li>A</li>
 <li>C</li>
</ol>
  1. B
  2. A
  3. C
