Outros elementos HTML

Line breaks

Os line breaks(quebra linhas),definidos por <br>, são tags que servem para quebrar uma linha num texto

O texto escrito após esta tag será exibido noutra linha

Exemplo de um line break num texto (com resultado)

<p>Exemplo de <br> um line break. </p>

Buttons

Os buttons são utilizados para ativar eventos do HTML, e são definidos pela tag <button> em que o texto escrito entre as suas tags é o texto que será exibido no botão.

Exemplo de um button (com resultado)

<button>Texto do botão</button>
Nota:

Os botões serão explicados com mais pormenor no javascript.
