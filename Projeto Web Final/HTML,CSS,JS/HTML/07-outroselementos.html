Elementos HTML

Outros elementos HTML

Title

Como tinha sido falado na introdução, o elemento <title> é utilizado para definir o titulo do documento(cabeçalho da página) do HTML.

Este elemento,ao contrário dos que foram falados até agora, é colocado no <head> dado que o elemento title é um elemento que oferece metainformação.

Exemplo de um title

<title>Titulo da Pagina</title>

Hyperlink

O Hyperlink,definido por <a>, é utilizado para colocar um link para uma página dentro do HTML.

O hyperlink necessita de um atributo href(em que o valor é o URL da página que pretende usar no link) para poder funcionar.

Exemplo de um hyperlink (com resultado)

<a href="https://www.google.pt" target ="_blank">Clique aqui para ir para a página do Google</a>
Clique aqui para ir para a página do Google
Nota:

O atributo target="_blank" serve para,quando o utilizador carregar no link,abrir um novo separador no browser.

Line breaks

Os line breaks(quebra linhas),definidos por <br>, são tags que servem para quebrar uma linha num texto

O texto escrito após esta tag será exibido noutra linha

Exemplo de um line break num texto (com resultado)

<p>Exemplo de <br> um line break. </p>

um line break.

Non-breaking space

O non-breaking space,definido por&nbsp; é uma entidade que serve para criar um espaço onde esta entidade estiver escrito num texto.

Exemplo de um non-breaking space num texto (com resultado)

<p>Exemplo de &nbsp um non-breaking space. </p>

Exemplo de   um non-breaking space.

Preformatted text

O preformatted text(texto pré-formatado),definido por <pre>, servem criar texto em que não é necessário utilizar line breaks ou non-breaking space,porque se der uma quebra de linha ou um espaço no texto em si, ele será registado com este elemento.

Exemplo de um preformatted text (com resultado)

<pre>Exemplo de um texto
pré-formatado em que não é necessário
line breaks nem non-breaking space. </pre> 
Exemplo de um texto 
pré-formatado em que não é necessário
line breaks nem     non-breaking space.

Buttons

Os buttons são utilizados para ativar eventos do HTML, e são definidos pela tag <button> em que o texto escrito entre as suas tags é o texto que será exibido no botão.

Exemplo de um button (com resultado)

<button>Texto do botão</button>
Nota:

Os botões serão explicados com mais pormenor no javascript.
