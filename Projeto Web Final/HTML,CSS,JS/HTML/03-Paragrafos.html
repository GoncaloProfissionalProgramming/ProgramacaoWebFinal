HTML Paragraph

A tag paragraph(parágrafo) é utilizado para criar texto no HTML na forma mais basica e é uma boa tag a utilizar para quem está a começar HTML

Este elemento é definidos por <p> e o conteudo dentro das tags é o texto a ser definido para o paragrafo.
