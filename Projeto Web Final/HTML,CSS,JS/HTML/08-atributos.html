Atributos HTML

Como tinha sido explicado na introdução do HTML, os atributos são informação adicional para um elemento do HTML, e costumam vir no formato nome = "valor".

Nesta parte será abordado atributos simples que poderá utilizar.

Atributo title

O atributo title serve para exibir um texto quando um utilizador arrastar o rato para cima do elemento com esse atributo.

Este atributo pode ser utilizado por qualquer elemento.

Exemplo do atributo title(com resultado)

<p title="Sou um atributo titulo">Exemplo</p>

Exemplo

Se arrastar o rato para cima do parágrafo exemplo, é exibido o texto escrito no atributo title(neste caso sendo 'Sou um atributo titulo').

Atributos width e height

Os atributos width e height servem para definir o comprimento (width) e a altura(height) de um elemento HTML.

Estes atributos só funcionam em elementos que suportem dimensões diferentes(imagens como foi exibido na parte anterior).

Exemplo dos atributos width e height (com resultado)

<img src="analysisBlack16.png" width="50" height="50"></img>

Nota: os valores de width e height vêm em pixeis,como no exemplo pode ser visualizado, a imagem ocupa 50x50(50 de altura e 50 de comprimento) pixeis no resultado.

Atributo src

O atributo src permite ir buscar um ficheiro externo a ser utilizado pelo elemento.

O valor a ser indicado para este atributo é a localização do ficheiro relativo a onde o ficheiro HTML esta localizado.

Como por exemplo, um ficheiro HTML encontra-se numa pasta principal e uma imagem encontra-se numa pasta img dentro da pasta principal.

Para o ficheiro HTML utilizar essa imagem para um elemento img,indica-se primeiro o nome da pasta, e depois o nome da imagem a ser utilizado,como por exemplo.

<img src="img/analysisBlack16.png" ></img>

Atributo alt

O atributo alt serve para exibir o valor desse atributo quando o elemento não consegue exibir o ficheiro externo a ser utilizado.

Como por exemplo se a imagem analysisBlack16.png não existir e tentar-mos buscar essa imagem no HTML, será exibido o texto do alt.

<img src="img/analysisBlack16.png" alt="Esta imagem não existe"></img>

Atributo style

O atributo style permite utilizar propriedades CSS para dar estilo a um elemento(mudar cor,mudar a fonte e o tamanho do texto,etc.).

As propriedades do atributo contem uma chave com um valor(exibido na forma chave:valor) e podem ser separados com ;

Exemplo do atributo style(com resultado)

<p style="color:red;text-align:center">Exemplo</p>

Exemplo

As propriedades deste atributo serão falados no tópico CSS dado que todos os atributos do atributo style encontram-se lá.
