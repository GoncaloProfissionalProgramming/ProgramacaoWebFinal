HTML Table

As tables(tabelas) são elementos para conter outros elementos em colunas e linhas para esses elementos estarem organizados em uma tabela.

Para a criação de tables, é necessário 3 tags para a table funcionar:

A tag <table> que representa a table em si.

A tag <tr> que representa uma linha da table.

A tag <td> que representa uma coluna da table.

Os tags da table vêm nesta ordem,apesar de poder existir vários elementos <tr> dentro da <table> e <td> dentro do <tr> para determinar o número de linhas e colunas da table.

Exemplo de uma Table(com resultados)

<table border="1">
 <tr>
  <td>Linha 1 Coluna 1</td>
  <td>Linha 1 Coluna 2</td>
 </tr>
 <tr>
  <td>Linha 2 Coluna 1</td>
  <td>Linha 2 Coluna 2</td>
 </tr>
</table>
Linha 1 Coluna 1 Linha 1 Coluna 2
Linha 2 Coluna 1 Linha 2 Coluna 2
Nota:

Neste exemplo foi utilizado o atributo border = "1" na table para desenhar linhas nos limites da tabela.
