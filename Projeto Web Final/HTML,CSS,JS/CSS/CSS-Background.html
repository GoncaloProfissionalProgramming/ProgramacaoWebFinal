Background CSS

As propriedades de background em CSS são usadas para definir efeitos nos elementos. Existem 5 propriedades que são:
  • Background-color
  • Background-image
  • Background-repeat
  • Background-attachment
  • Background-position

    • Background Color

    Podemos definir a cor que vai estar por trás de algo

    Exemplo

    <h3 style="background-color:blue;">Background-color Azul</h3>
    <p style="background-color:green;">Background-color Verde</p>

    Background-color Azul

    Background-color Verde

    Background-image

    Podemos colocar uma imagem no background

    Exemplo

    body {
    background-image: url("css.jpg");
    }

    Background-repeat

    O background-repeat serve para repetir o background

    Exemplo

    body {
    background-image: url("css.jpg");
    background-repeat:repeat;
    }
