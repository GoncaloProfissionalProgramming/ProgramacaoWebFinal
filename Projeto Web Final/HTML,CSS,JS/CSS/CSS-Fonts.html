Fontes CSS

A propriedade Font em CSS permite definir a familia da fonte, se é bold ou não, o tamanho da letra e o estilo do texto.

CSS Font Families

Em CSS existem dois tipos de familias de fontes que são:

  • Família Genérica- Grupo de fontes que são parecidas umas com as outras, por exemplo: "Serif" ou "Monospace".
  • Família de Fontes- É uma fonte especifica, por exemplo: "Arial" ou "Calibri".

Font Family

A fonte de um texto é definida com a propriedade font-family.

Esta propriedade deve conter o nome de várias fontes, assim se o browser não conseguir reconhecer a primeira fonte irá tentar a segunda e por aí em diante.

Começa-se pela fonte que se quer utilizar e termina-se por uma genérica para que o browser escolha uma fonte parecida na família genérica, se não estiver disponível outra fonte.

Nota: Se o nome da fonte for mais que uma palavra então esta deve estar entre aspas, por exemplo: "Times New Roman".

Mais que uma fonte deve ser especifica entre vírgulas.

Exemplo

h1 {
font-family: "Times New Roman", Arial, Monospace;
}

Font Style

A propriedade CSS Font-Style é mais usada para definir texto em itálico.

  • Normal- O texto é mostrado como normal
  • Itálico- O texto é mostrado a itálico.
  • Obliquo- O texto é "inclinado" (muito semelhante ao itálico, mas menos usado)

Exemplo

<p style="font-style: normal;">Texto normal</p>
<p style="font-style: italic;">Texto a itálico</p>
<p style="font-style: oblique;">Texto a Oblíquo</p>

Font Size

A propriedade CSS Font-Size é usada para definir o tamanho do texto.

Existem 3 formas de definir o tamanho do texto:

  • Percentagem (%)
  • Em (em)
  • Pixeis (px)

Exemplo

<p style="font-size: 120%;">Texto definido com Percentagem</p>
<p style="font-size: 5em;">Texto definido com em</p>
<p style="font-size: 25px;">Texto definido com pixeis</p>

Font Weight

A propriedade CSS Font-Weight é usada para definir o a grossura do texto.

Existem 3 formas de definir a grossura do texto:

  • Normal
  • Bold
  • Lighter
  • Por números

Exemplo

<p style="font-weight: normal;">Texto normal</p>
<p style="font-weight: bold;">Texto a bold</p>
<p style="font-weight: lighter;">Texto em lighter</p>
<p style="font-weight: 600;">Texto por numeros</p>
