Borders CSS

A propriedade CSS Border permite-nos especificar o style, largura e cor de um elemento.

Border-Style

A propriedade Border-Style especifica qual tipo de border mostra.
Os valores abaixo são permitidos:

  • <dotted> é uma borda com pontos.
  • <dashed> é uma borda com traços.
  • <solid> é uma borda de uma linha única.
  • <double> é uma borda dupla.
  • <groove> define uma borda 3D, o efeito depende da cor da borda.
  • <ridge> define uma borda 3D com sulcos, o efeito depende da cor da borda.
  • <inset> define uma borda em 3D, o efeito depende do valor da cor da borda, com um efeito de fora para dentro.
  • <outset> define uma borda em 3D, o efeito depende do valor da cor da borda, com um efeito de dentro para fora.
  • <none> não tem borda.
  • <hidden> é uma borda escondida.
A propriedade Border-Style pode ter um de quatro valores(borda de cima, borda direita, borda de baixo e borda da esquerda).

Exemplo

p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}
p.mix {border-style: dotted dashed solid double;}

Border-Width

A propriedade Border-Width especifica a largura das quatro bordas.
A largura pode ser definida para um tamanho específico (em px, pt, cm, in, etc.) ou usando um de três valores padrão: fino, médio ou grosso.
A propriedade Border-Width pode ter um de quatro valores (borda de cima, borda direita, borda de baixo e borda da esquerda).

Exemplo

border{
  border-style: solid;
  border-width: 5px;
}

Border com 5px

Border-Color

A propriedade Border-Color é usada para definir a cor das quatro bordas.
A cor pode ser definida por:

  • Nome - especifica um nome de cor, como "red"
  • Hex - especifica um valor hexadecimal, como "#ff0000"
  • RGB - especifica um valor RGB, como "rgb(255,0,0)"
  • Transparent
Se a Border-Color não estiver definida esta irá ficar com a cor elemento em que estiver inserida.
A propriedade Border-Color pode ter um de quatro valores (borda de cima, borda direita, borda de baixo e borda da esquerda).

Exemplo

border{
  border-style: solid;
  border-color: #0095D9;
}
