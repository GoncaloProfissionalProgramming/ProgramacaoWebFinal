CSS

Introdução ao CSS

O que é CSS?

CSS significa Cascading Style Sheets, esta linguagem é principalmente usada para costumizar o design de um website.

Como utilizar CSS?

3 Formas de utilizar CSS

1-Ficheiro externo por exemplo "style.css"

Style.css

body {
background-color: #d0e4fe;
}

h1 {
color: orange;
text-align: center;
}

p {
font-family: "Times New Roman";
font-size: 20px;
}

2-Dentro de um ficheiro HTML usando a tag <style></style>

index.html

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<head>

<style>

h1 {
   color: orange;
   text-align: center;
}

</style>

</head>

<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

3-Dentro de um elemento HTML usando o style="";

index.html

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<body>

<h1 style="color:blue;">Este texto está a azul</h1>
<p>This is a paragraph.</p>

</body>
</html>

Syntax CSS

A Syntax CSS divide-se em dois, ou seja para podermos editar um elemento html temos de ter um selector e uma declaração que se divide em propriedade e valor. Por exemplo:

h1 { color:blue;} aqui h1 é um selector, {color:blue;} é uma declaração, color é a propriedade, blue é o valor.

Selectores CSS

Exitem 4 tipos de Selectores em CSS

1-Selector de elemento

Exemplo

p {
text-align: center;
}

2-Selector por id

Exemplo

#div {
position:absolute;
text-align: center;
}

3-Selector por class

Exemplo

.div {
text-align: center;
}

4-Selector por grupo

Exemplo

h1,h2 {
color:blue;
}

Comentar código CSS

Para comentar linhas de código em CSS usa-se /* linha de código */

Exemplo

/* p {
text-align: center;
} */
