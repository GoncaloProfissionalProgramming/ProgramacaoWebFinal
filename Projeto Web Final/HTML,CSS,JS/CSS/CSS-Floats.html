CSS

Floats CSS

A propriedade Float em CSS é usada para posicionar e formatar conteúdo.
A propriedade float pode ter um de quatro valores que são:
  • Left- Flutua á esquerda do container
  • Right- Flutua á direita do container
  • none- Não flutua
  • inherit- Herda o float do container

CSS float:right

Exemplo

<p style="float:right;">Texto á direita</p>

Texto á direita



CSS float:left

Exemplo

<p style="float:left;">Texto á esquerda</p>

Texto á esquerda



CSS float:none

Exemplo

<p style="float:none;">Texto sem float</p>

Texto sem float

CSS float:inherit

Exemplo

<p style="float:inherit;">Texto com float em inherit</p>

Texto com float em inherit
