    Uma família quer listar os seus nomes,parentalidade e idade numa tabela.

    Crie uma tabela em que:

    • Tenha linhas com o valor de 1 na tabela.
    • A primeira linha tenha cabeçalhos com o texto nome, parentalidade e idade
      • Utilize o elemento <th> para criar cabeçalhos nas tabelas
    • Adicione os valores na coluna certa de cada membro da família por cada linha após a primeira
      • O primeiro membro é o José que é o pai da família e tem 48 anos.
      • O segundo membro é a Maria que é a mae da família e tem 45 anos.
      • O terceiro membro é o Ricardo que é o filho da família e tem 14 anos.

    Crie uma página HTML em que:

    • Apresente um heading h2 com o nome Miguel
      • Apresente um paragraph com o número 10
      • Title da pagina com o titulo de "Exercicio"

      Solução

      • Para criar um heading h2 com um nome, utilizamos a tag de abertura <h2>, escrevemos Miguel após a tag e usamos a tag de fecho </h2> para fechar o elemento

        • O elemento deve de ficar nesta forma

        <h2>Miguel</h2>

        Com o resultado desta forma

        Miguel

      • Para criar um paragraph com o número 10, tal como no heading, criamos a tag de abertura <p> primeiro, depois escrevemos 10 e depois fechamos com a tag de fecho </p>
        • <p>10</p>

        E o resultado irá aparecer desta forma

        10

      • Para criar um title com o titulo de "Exercicio", precisamos de criar dentro do <head> do documento, dado que se trata de uma tag de metainformação.

        • Criamos a tag de abertura <title>, escrevemos Exercicio e fechamos com a tag de fecho </title>

        <title>Exercicio</title>

        Deverá de aparecer no titulo da pagina o texto "Exercicio"

        Como o documento deverá estar.

        <!DOCTYPE html>
        <html>

         <head>

          <title>Exercicio</title>

         </head>

         <body>

          <h2>Miguel</h2>

          <p>10</p>

         </body>

        </html>

      Crie uma pagina HTML em que:

      • Tenha uma imagem com largura igual a 50 e altura igual a 100 e que exiba o texto "imagem não encontrada" quando o elemento não consegue encontrar a imagem.
        • Pode utilizar a imagem que quiser
      • Tenha 2 listas de cada tipo, em que na lista com pontos que liste 1,2 e 3 e na lista com numeração que liste a,b e c.

      Solução

      • Para a criação da imagem com as especificações do exercício, é necessário utilizar os atributos src,width,height e alt

        • No atributo src, coloca-se o endereço onde a imagem se encontra relativamente á raiz do documento HTML(se a imagem tiver na pasta do documento,simplesmente indique o nome da imagem).

        No atributo width, coloca-se o valor da largura(ou seja 50) e no atributo height coloca-se o valor da altura(100).

        No atributo alt,coloca-se o texto "imagem não encontrada" dado que o atributo alt serve para exibir algo quando o documento não consegue aceder á imagem.


        O elemento deverá de ter esta forma.

        <img src="imagem.png" width="50" height="100" alt="imagem não encontrada"></img>
      • Para a criação da lista com pontos, utilizamos o elemento <ul> para criar a lista.

        • Como queremos listar os números 1,2 e 3, criamos o elemento <li> para cada número(cada número fica dentro das tags de um elemento li )

      <ul>

       <li>1</li>

       <li>2</li>

       <li>3</li>

      </ul>

      O resultado deverá ser este

      • 1
      • 2
      • 3

      Para a criação da lista com numeração,utilizamos o elemento <ol> para criar a lista.

      Utilizamos o mesmo método para listar as letras a,b e c,dado que cada item da lista utiliza também o elemento criamos o elemento <li>(basta trocar os números pelas letras)

      <ol>

       <li>a</li>

       <li>b</li>

       <li>c</li>

      </ol>

      O resultado deverá ser este

      1. a
      2. b
      3. c

      Uma família quer listar os seus nomes,parentalidade e idade numa tabela.

      Crie uma tabela em que:

      • Tenha linhas com o valor de 1 na tabela.
      • A primeira linha tenha cabeçalhos com o texto nome, parentalidade e idade
        • Utilize o elemento <th> para criar cabeçalhos nas tabelas
      • Adicione os valores na coluna certa de cada membro da família por cada linha após a primeira
        • O primeiro membro é o José que é o pai da família e tem 48 anos.
        • O segundo membro é a Maria que é a mae da família e tem 45 anos.
        • O terceiro membro é o Ricardo que é o filho da família e tem 14 anos.

      Solução

      • Para a criação da tabela utilizamos o elemento <table>. Adicionamos o atributo border="1" neste elemento para colocar-mos as linhas na tabela.
      • Dentro do elemento table criamos a primeira linha da tabela com o elemento <tr> e adiciona-mos dentro desse elemento 3 elementos <th> para criarmos cabeçalhos dentro da primeira linha.

        • Por cada <th> adicionamos dentro o texto que é necessário adicionar aos cabeçalhos(nome,parentalidade e idade).

        <tr>

         <th>nome</th>

         <th>parentalidade</th>

         <th>idade</th>

        </tr>
      • Criamos mais 3 linhas para cada membro da família,em que utilizamos o elemento <td> para escrever os valores de cada membro por ordem da coluna(primeiro nome,segundo parentalidade,terceiro idade).
        • <tr>
         <td>jose</td>
         <td>pai</td>
         <td>48</td>
        </tr>
        <tr>
         <td>maria</td>
         <td>mãe</td>
         <td>45</td>
        </tr>
        <tr>
         <td>ricardo</td>
         <td>filho</td>
         <td>14</td>
        </tr>

        A resultado da tabela deverá de ser este.

        nome parentalidade idade
        jose pai 48
        maria mãe 45
        ricardo filho 14
