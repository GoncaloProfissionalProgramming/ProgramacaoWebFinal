Solução

Um computador é constituído por dois sistemas: o hardware e o software. O hardware designa os componentes físicos. O software designa o código ou os programas que transformam o computador numa máquina operacional.

Os programas são constituídos por instruções, codificadas em código binário, instruções essas que são interpretadas e executadas pelo processador ou CPU.

Tudo o que o qualquer computador faz é executar programas, e começa a executar programas no momento em que é ligado. Esses programas de arranque estão armazenados em memória ROM (memória só de leitura) e uma vez que estão permanentemente gravados no hardware, esses programas recebem a desingação genérica de firmware.

Os programas de arranque contêm um conjunto de rotinas básicas de ligação a periféricos e executam um conjunto inicial de operações de auto-diagnóstico (POST) e iniciam o carregamento do sistema operativo para a memória.

Solução

A arquitectura de um computador constitui a interface lógica que o sistema apresenta perante os seus utilzadores. No caso do processador, a arquitectura é representada por um determinado instruction set e por um conjunto de funcionalidades internas a que os programadores podem ter acesso.

O conceito de estrutura e organização diz respeito ao modo como o sistema é internamente estruturado e ao modo como as funcionalidades são implementadas. De um modo geral, a organização e a estrutura têm a ver com a tecnologia usada para implementar as funcionalidades da arquitectura.

As linhas fundamentais da arquitectura dos computadores modernos foram definidas nos anos 50 do século XX pelo cientista Von Neumann. Essa arquitectura é conhecida por “von Neumann architecture” e introduz o conceito de stored program: isto é o conceito de uma máquina que pode executar diferentes programas, que deverão ser armasenados na memória e executados na ALU (Arithmetic and Logic Unit).

Solução

A informação digital é representada através de bits. O bit é a unidade elementar de informação. Do ponto de vista físico, um bit pode ser qualquer dispositivo físico que, em cada momento, possa assumir um de dois estados diferentes. Do ponto de vista lógico, os dois estados de um bit são representados pelos dígitos binários 0 e 1.

Uma sequência de n bits permite representar 2 elevado a n estados diferentes. Usando sequências de bits pode representar-se qualquer tipo de informação digital: números, letras, imagens e som.
