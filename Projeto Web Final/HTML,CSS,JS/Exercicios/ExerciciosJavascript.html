Utilize uma caixa Alert para exibir:

  • Uma varíavel x do tipo String que tenha o valor de "Exercicio".
  • O comprimento(length) da varíavel x.
  • A soma entre uma variável y(com valor 5) e a variável z(com valor 4).

Solução

  • Para criar uma caixa Alert, utiliza-se dentro do <script> o metodo alert(x);, em que x pode ser tanto uma variável como uma String.
  • Cria-se a variável x desta forma var x = "Exercicio" e ela será do tipo string porque foi atribuido uma string á variável.
  • Para determinar o comprimento da variável utiliza-se o metodo length, que se encontra dentro da variável x(ou seja, no script escreve-se x.length para obter o comprimento da variável x).
  • Para exibir a soma das variáveis, cria-se uma variável extra (para esta solução chamo a).

    • Atribui-se a essa variável var a = y + z e coloca-se a variável a no alert para ser exibido.

  • Como o script deve de estar(pode ter aspeto diferente no seu lado mas desde que tenha os resultados pedidos é permitido).
    • <script>
     var x = "Exercicio"
     var y = 5;
     var z = 4;
     var a = 5 + 4;
     alert(x);
     alert(x.length);
     alert(a);
    </script>

Tendo em conta as seguintes variáveis:

var a = 10
var b = 4
var c = 2
var d = 2
  • Crie um if que verifique se a variável a é igual á b.(Use um alert para avisar se é igual ou não).
  • Crie um while que enquanto a variável d não for igual á variável a, o valor da variável d aumenta.
  • Crie um switch para b + c em que:
    • Se o valor for 6, o valor de a é igual a 2
    • Se for 4, o valor de a é igual a 6
    • Se for 8, o valor de a é igual a 12
    • Coloque o resultado de a num alert

Solução

  • Após criar as variáveis, cria-se um if nesta forma if(a == b) e abre-se chavetas {}

    • Dentro das chavetas, crie um alert que avise que as variáveis são iguais alert("variáveis são iguais");.

    Depois das chavetas, crie um else (com chavetas) e dentro das chavetas escreva um alert que avise que as variáveis são diferentes alert("variáveis são diferentes");.

  • Crie um while nesta forma while(d != a) e abra chavetas e dentro das chavetas, insira d++; para ir adicionando 1 valor á variavel d enquanto a for diferente de d.
  • Crie um switch desta forma switch(b + c) e abra chavetas

    • Dentro das chavetas, crie um case para cada linha pedida no exercício

    Por exemplo para a primeira linha, o case é 6,coloca-se a = 2.

    Após o switch, coloque um alert a exibir a variavel a.

  • O script deve de ter mais ou menos esta forma:
    • <script>
     var a = 10;
     var b = 4;
     var c = 2;
     var d = 2;

     if(a == b) {
       alert("variáveis são iguais");
     }
     else {
       alert("variáveis são diferentes");
     }
     while(a != d) {
       d++;
     }
     switch(b + c) {
       case 6:
        a = 2
        break;
       case 4:
        a = 6
        break;
       case 8:
        a = 12
        break;
     }
     alert(a);
    </script>

Crie uma caixa alert com o valor do volume de um cilindro,com 3 casas decimais, sabendo que:

raio = 5
altura = 10
  • Crie um object cilindro com as propriedades altura e diametro
  • Crie dentro do objeto um metodo que retorna o valor do volume do cilindro.
    • Utilize a formula V = 3.14 * r² * h , em que V é o volume, r é o raio e h é a altura.
    • utilize parsefloat(valor do volume).toFixed(3) para o valor ter 3 casas decimais
    • Utilize este método para exibir a mensagem na caixa alert.

Solução

Para criar o objeto cilindro, começamos por criar 2 variveis altura(h) e raio(r) com os respetivos valores e uma variável do tipo object var cilindro = new Object();

Fazemos cilindro.altura = h e cilindro.diametro = r * 2 para definir as propriedades altura e diametro.

Fazemos cilindro.Volume = function() para criarmos o metodo onde vamos retornar o valor do volume, abrimos chavetas após a declaração.

Dentro das chavetas, criamos uma variavel(volume) para calcular o volume do cilindro usando a formula do exercício var volume = 3.14 * pow(this.diametro /2),2) * this.altura (this serve para referenciar a propriedade do proprio objeto e o metodo pow serve para calcular o valor do raio ao quadrado).

Coloca-se volume = parsefloat(volume).toFixed(3) para que o valor esteja arredondado para 3 casas decimais.

Finalmente fora da função escreve-se a alert box a chamar o metodo volume desta forma alert(cilindro.Volume());

O script deverá de ter mais ou menos esta forma:

<script>

 var cilindro = new Object();
 var h = 10;
 var r = 5;
 cilindro.altura = h
 cilindro.diametro = r * 2

 cilindro.Volume = function() {
  var volume = 3.14 * pow(this.diametro /2),2) * this.altura;
  volume = parsefloat(volume).toFixed(3);
  return volume;
 };
 alert(cilindro.Volume());

</script>
