Altere o elemento do paragrafo da forma que:

  • Altere a cor do texto do paragrafo para vermelho dentro do elemento
  • Altere o background do paragrafo para azul dentro da tag <style>
    • Lembre-se que a tag <style> é necessária ser criada dentro do head.

Crie uma border no paragrafo em que:

  • É uma borda com traços.
  • Tenha uma largura de valor 5 pixeis.
  • Tenha uma cor azul.
    • Pode utilizar qualquer dos métodos anteriores para adicionar os atributos ao elemento.

Altere o texto do paragrafo na forma que:

  • O seu texto fique posicionado á esquerda(flutuar á esquerda).
  • A font do texto seja "Times New Roman".
  • O tamanho do texto seja 80% do tamanho de um texto normal.
  • O texto esteja em itálico.

Altere o elemento do paragrafo da forma que:

  • Altere a cor do texto do paragrafo para vermelho dentro do elemento
  • Altere o background do paragrafo para azul dentro da tag <style>
    • Lembre-se que a tag <style> é necessária ser criada dentro do head.

Solução

  • Para alterar a cor do texto dentro do elemento do paragrafo, utilizamos o atributo style.

    • Dentro do style, utilizamos a chave color e definimos que color:red.

  • Para alterar o background do paragrafo, utilizamos o atributo background-color.

    • Dentro da tag <style>, defenimos o elemento que queremos colocar o atributo, que neste caso é o elemento paragrafo(p).

    Após definir o elemento, abrimos chavetas {} e colocamos dentro o atributo background-color:blue;.

  • O Documento deverá de ter esta forma.
    • <!DOCTYPE html>
    <html>

     <head>
      <style>
       p{
        background-color:blue;
       }
      </style>
     </head>

     <body>
      <p style="color:red">Exercicio</p>
     </body>

    </html>

Crie uma border no paragrafo em que:

  • É uma borda com traços.
  • Tenha uma largura de valor de 5 pixeis.
  • Tenha uma cor azul.
    • Pode utilizar qualquer dos métodos anteriores para adicionar os atributos ao elemento.

Solução

  • Para criar uma borda com traços, utilizamos a chave border-style:dashed;.
  • Para definir a largura da borda, utilizamos a chave border-width:5px;
  • Para definir a cor da borda, utilizamos a chave border-color:blue;
  • No final, o documento deverá de ter uma destas 2 formas:
    • <!DOCTYPE html>
    <html>

     <head>
      <style>
       p{
        border-style:dashed;
        border-width:5px;
        border-color:blue;
       }
      </style>
     </head>

     <body>
      <p>Exercicio</p>
     </body>

    </html>
    <!DOCTYPE html>
    <html>

     <head>
     </head>

     <body>
      <p style="border-style:dashed;border-width:5px;border-color:blue;">Exercicio</p>
     </body>

    </html>

Altere o texto do paragrafo na forma que:

  • O seu texto fique posicionado á esquerda(flutuar á esquerda).
  • A font do texto seja "Times New Roman".
  • O tamanho do texto seja 80% do tamanho de um texto normal.
  • O texto esteja em itálico.

Solução

  • Para que o texto fique posicionado á esquerda, utilizamos a chave float:right;.
  • Para que o tamanho do texto seja 80% do normal, utilizamos a chave font-size:80%;.
  • Para que o texto fique em itálico, utilizamos a chave font-style:italic;.
  • No final, o documento deverá de ter uma destas 2 formas:
    • <!DOCTYPE html>
    <html>

     <head>
      <style>
       p{
        float:right;
        font-size:80%;
        font-style:italic;
       }
      </style>
     </head>

     <body>
      <p>Exercicio</p>
     </body>

    </html>
    <!DOCTYPE html>
    <html>

     <head>
     </head>

     <body>
      <p style="float:right;font-size:80%;font-style:italic;">Exercicio</p>
     </body>

    </html>
